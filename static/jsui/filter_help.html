Common filter clauses

ClauseMeaning
[src|dst] host <ip>Matches a host as the IP source, destination, or either
[src|dst] net <network>/<len>Matches packets to or from an endpoint residing in network
ether [src|dst] host <mac>Matches a host as the Ethernet source, destination, or either
[tcp|udp] [src|dst] port <port>Matches TCP or UDP packets sent to/from port
[tcp|udp] [src|dst] portrange <p1>-<p2>Matches TCP or UDP packets to/from a port in the given range
(ether|ip|ip6) proto <protocol>Matches an Ethernet, IPv4, or IPv6 protocol
(ether|ip) broadcastMatches Ethernet or IPv4 broadcasts
(ether|ip|ip6) multicastMatches Ethernet, IPv4, or IPv6 multicasts

Combining clauses

CombinationMeaning
<clause1> and <clause2>Matches when both clauses match
<clause1> or <clause2>Matches when one or both clauses match
not <clause1>Matches when clause does not match

See www.tcpdump.org for more information.
