Editor info

This editor is part of the topology management tool (ToMaTo). ToMaTo is part of the German-Lab Research Project.

Browser support

Actively supported browsers

ChromeFully supported, in fact this browser is used during development.
FirefoxFully supported.

Other browsers

OperaShould work. Opera has an issue that clicking with the ctrl-key held zooms into the page. To circumvent this issue the alt-key can be used instead.
Internet ExplorerMight work. Internet Explorer has its own interpratation of web standards. Its implementation differs from the other browsers in so many ways that it is nearly impossible to develop for open browsers and IE at the same time. Better get a sane browser.
